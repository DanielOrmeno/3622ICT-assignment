The Javascript libraries used in the creation of the DMS website are as follows

  • AngularJs
  • Angular.Ui
  • Ng Facebook
  • Angular Bootstrap Lightbox
  • Facebook API Libary
  • Bootstrap
  • Webpack
  • Lodash

All javascript written for this assignment conforms to the most up to date JavaScript specification (EcmaScript 6+) however no current browser supports the litany of new and exciting features in EcmaScript 6. To enable myself to use the newest standard I've used WebPack to transpile all ES6 JavaScript to ES5 so that it may be run in modern browsers
